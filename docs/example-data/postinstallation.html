Steel 1.0

Hello, and Welcome to Steel.

Steel is an easy-to-use framework built from the ground up to give you the tools you need to build your application.

Read the docs

Why am I seeing this page?

This is the default MVC for the Steel Framework. It is used in place of an ugly "Hello World!" screen :)

Quick-Start

  • To start making your app (hopefully you've read the docs and know how to make an app,) you need to configure Steel. Open up /classes/Steel/Settings.php and configure the following to get started:
  • If you have files to include into Steel, set ['steel']['autoinclude'] to true and place your PHP files in the /include directory.
  • If you'll be using an Application to hook into Steel, set ['steel']['useApplication'] to true and make sure your ['steel']['application'] is properly entered. Check the wiki for more info.
  • If you plan on using sessions, set ['steel']['useSessions'] to true.
  • If you plan on using the integrated localizer, set ['steel']['useLang'] to true.
  • The ['resources']['css'] and ['resources']['js'] arrays are arrays that autoload your resources. This requires Subresource Integrity. If you want to add your own, just follow the format of the entries already there.
  • Change ['general']['host'] to the domain name you want pages to redirect to. EX: {['general']['host']}/error/404
  • Configure ['database'] to meet your database criteria if you plan on using the built in database system.
  • Lastly, when you're ready, set ['steel']['postinst'] to false. That will disable this page and let you get to work!
